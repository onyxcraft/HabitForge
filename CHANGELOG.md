# Changelog

All notable changes to HabitForge will be documented in this file.

## [1.0.0] - 2026-03-25

### Added
- Initial release of HabitForge
- Create and manage habits with custom icons, colors, and frequencies
- One-tap habit completion with satisfying animations
- Streak tracking with milestone fire animations (7, 30, 100+ days)
- GitHub-style calendar heatmap for visualizing habit history
- Comprehensive statistics: current streak, longest streak, total completions, success rate
- Home screen widgets:
  - Habit Checklist widget (small/medium sizes)
  - Top Streak widget (small size)
- Customizable reminder notifications per habit
- Habit groups and categories for organization
- Full dark mode support
- iPad-optimized layouts
- SwiftUI + MVVM architecture
- SwiftData for local persistence
- No external dependencies
